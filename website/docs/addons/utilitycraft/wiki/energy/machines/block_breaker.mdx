## Blocks It Cannot Break

For balance reasons, the **Block Breaker** cannot break the following blocks:

- Bedrock  
- End Portal Frame  
- End Portal  
- Nether Portal  
- Reinforced Deepslate  
- Command Blocks *(Normal, Chain, Repeating)*  

---

### How to Use

1. Place the Block Breaker facing the block you want to break.  
2. Ensure it has enough stored energy (**800 DE per operation**).  
3. *(Optional)* Add **Speed Upgrades** to increase breaking speed.
