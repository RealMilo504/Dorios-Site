The **Induction Anvil** is a UtilityCraft machine that repairs damaged items using **Dorios Energy (DE)** instead of experience,  
making it a reliable alternative to traditional anvils.

---

### Machine Information

| Setting | Value |
|----------|--------|
| **Energy Cap** | 16,000 DE |
| **Energy Cost** | 5 DE per durability point |
| **Base Repair Speed** | 1 durability per tick |

| Speed Level | Ticks per Repair Cycle | Durability Repaired per Second |
|--------------|------------------------|--------------------------------|
| 0 (No Upgrades) | 80 | 4 durability/sec |
| 1 | 40 | 8 durability/sec |
| 2 | 20 | 16 durability/sec |
| 3 | 10 | 32 durability/sec |
| 4 | 5 | 64 durability/sec |

---

### How It Works

- Consumes **5 DE** per durability restored.  
- Repairs the inserted item automatically using stored energy.  
- Base repair rate: **1 durability per tick**.  
- Accepts **Speed Upgrades** to increase repair rate.

---

### How to Use

1. Place the **Induction Anvil** in your world.  
2. Insert a **damaged item** into the repair slot.  
3. Ensure it has enough stored energy (**5 DE per durability point**).  
4. *(Optional)* Add **Speed Upgrades** to increase repair speed.
