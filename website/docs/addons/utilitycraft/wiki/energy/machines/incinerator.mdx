The **Incinerator** is a UtilityCraft machine that works as an electric furnace.  
It smelts ores, metals, blocks, and food using **Dorios Energy (DE)** instead of fuel.

---

### Machine Information

| Setting | Value |
|----------|--------|
| **Energy Cap** | 16,000 DE |
| **Energy Cost per Operation** | 800 DE |
| **Base Processing Rate** | 20 DE/t |

### How to Use

1. Place the Incinerator in your world.  
2. Insert an accepted item into the input slot.  
3. Ensure it has enough stored energy (**800 DE per operation**).  
4. (Optional) Add **Speed Upgrades** to increase smelting speed.
