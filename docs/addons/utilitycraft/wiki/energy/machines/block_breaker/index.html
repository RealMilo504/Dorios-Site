Block Breaker

The Block Breaker is a UtilityCraft machine designed to automate block breaking,
making it ideal for farms, mining setups, and automation systems.

Machine Information​

SettingValue
Energy Cap16,000 DE
Energy Cost per Operation800 DE

💡 Tip: The Block Breaker’s speed can be improved using Speed Upgrades,
allowing it to break blocks as fast as every 0.25 seconds.

Usage Examples​

InputOutput
Stone BlockDropped Stone
LogDropped Log Item

Crafting Recipe​

Blocks It Cannot Break​

For balance reasons, the Block Breaker cannot break the following blocks:

  • Bedrock
  • End Portal Frame
  • End Portal
  • Nether Portal
  • Reinforced Deepslate
  • Command Blocks (Normal, Chain, Repeating)

📌 With max Speed Upgrades, the Block Breaker can break a block every 0.25 seconds.

How to Use​

  1. Place the Block Breaker facing the block you want to break.
  2. Ensure it has enough stored energy (800 DE per operation).
  3. (Optional) Add Speed Upgrades to increase breaking speed.
