Infuser

The Infuser is a machine that combines base materials with an infusing item
(such as Redstone, Coal, or Dusts) to create new resources.

How to Use‚Äã

  1. Place the Infuser in your world.
  2. Insert the infusing item in the top slot (e.g., Redstone, Coal, Coal Dust).
  3. Insert the base material in the bottom slot (e.g., Iron Ingot).
  4. Ensure enough stored energy (800 DE per operation).
  5. (Optional) Add Speed Upgrades to increase processing speed.
